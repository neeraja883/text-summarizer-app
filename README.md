# 📝 Text Summarization App

A GenAI-powered text summarization app built with Python, Streamlit and OpenRouter API.

## Features
- 3 summary styles: Brief, Detailed, ELI5
- Clean Streamlit UI
- Powered by LLM via OpenRouter

## How to Run
1. Clone the repo
2. Create `.env` with your `OPENROUTER_API_KEY`
3. pip install -r requirements.txt
4. streamlit run app.py

## Tech Stack
- Python, Streamlit, OpenAI SDK, OpenRouter API
